# GPT AI Assistant

<div align="center">

[![license](https://img.shields.io/pypi/l/ansicolortags.svg)](LICENSE) [![Release](https://img.shields.io/github/release/memochou1993/gpt-ai-assistant)](https://GitHub.com/memochou1993/gpt-ai-assistant/releases/)

GPT AI Assistant is an application built using the OpenAI API and LINE Messaging API. After setup, you can chat with your own AI assistant directly via the LINE mobile app.

📰 News
2024-07-10: Version 4.9 now supports the gpt-4o OpenAI model. 🔥

2023-05-03: Version 4.6 adds support for the gpt-4 OpenAI model.

2023-03-05: Version 4.1 adds support for LINE audio messages and the whisper-1 OpenAI model.

2023-03-02: Version 4.0 now supports the gpt-3.5-turbo OpenAI model.
